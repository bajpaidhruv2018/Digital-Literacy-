# Stay Safe Online: A College Student's Checklist

1. **Verify Open Source Packages:** Always check the download counts and repository activity before installing npm or pip packages to avoid supply chain attacks.
2. **Nuke SMS 2FA:** Use a dedicated authenticator app (like Google Authenticator or Authy) instead of relying on easily compromised SMS verification.
3. **Commit and Push Daily:** Never leave important code sitting locally. Push to GitHub daily so a localized hardware failure or ransomware attack doesn't wipe your portfolio.
4. **Use a Password Manager:** Stop reusing the same password for your college portal, GitHub, and email. Let a manager generate and store complex keys.
5. **Beware of Phishing Links:** If an email from "college administration" or a "recruiter" demands urgent action via a link, verify the sender's actual email address first.
6. **Keep Environments Segmented:** Use tools like WSL (Windows Subsystem for Linux) or Docker to run development environments safely away from your main OS files.
7. **Financial Safety (UPI Tip 1):** Remember that your UPI PIN is ONLY required to send money. If a prompt asks for your PIN to "receive a refund" or "accept cash," it is a scam.
8. **Financial Safety (UPI Tip 2):** Never scan a random QR code to pay a vendor without manually verifying the merchant's registered name on your UPI app screen before authorizing.

### 🚨 Emergency Reporting Channels
If you are the victim of a cybercrime or financial fraud, report it immediately:
* **National Cyber Crime Portal:** [cybercrime.gov.in](https://cybercrime.gov.in)
* **Cyber Crime Helpline:** Call **1930** (Available 24x7)
